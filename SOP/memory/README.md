# Memory

This folder is the working memory of the estimating brain.

Use it to store:
- lessons learned from reviewed estimates,
- accepted benchmark refinements,
- recurring edge cases,
- estimate summaries worth reusing,
- and notes on what Claude should ask next time.

Low-risk updates can be added here automatically.

Examples of low-risk updates:
- adding a smart-lock lesson learned
- saving a new estimate summary
- promoting a clearly accepted benchmark note
- adding a reminder question for a category

Changes that alter core behavior should still be reviewable elsewhere:
- prompt rewrites
- schema changes
- benchmark policy changes
